vault child tokens

June 1, 2020

To continue from last time: we need to create some vault policy to allow terraform to create child tokens. Thanks to (tyrannosauraus-becks )[https://github.com/tyrannosaurus-becks], who suggests to add this policy to the github users that we have previously created:

path “auth/token/create” { capabilities = [“create”, “read”, “update”, “delete”, “list”] }

Huh, look at that, it worked. It’s nice when things work the first time! Now to rid our repos of secrets!
